# RTSP Middleware

Use node.js as middleware to translate RTSP into WebSocket

## 运行环境

- [ffmpeg](https://www.ffmpeg.org/download.html)
- [node.js](https://nodejs.org/en/download/)

## 使用

```bash
git clone <repo>
cd <repo>
npm install
```
